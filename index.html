Regra de Três

O que é Regra de Três?

A regra de três é um método matemático que permite encontrar um valor desconhecido quando temos três valores conhecidos de duas grandezas proporcionais.

Fórmula Básica:

a / b = c / x

Resolvendo para x: x = (c × b) / a

Tipos de Regra de Três:

Simples Direta

Quando duas grandezas aumentam ou diminuem na mesma proporção. Se uma aumenta, a outra também aumenta.

Exemplo: Se 5 kg de arroz custam R$ 20, quanto custam 10 kg? (Mais arroz = Mais caro)

Simples Inversa

Quando uma grandeza aumenta e a outra diminui na mesma proporção. Se uma aumenta, a outra diminui.

Exemplo: Se 3 operários fazem um trabalho em 12 dias, quantos dias levarão 6 operários? (Mais operários = Menos dias)

Passo a Passo:

  1. Identifique as duas grandezas envolvidas
  2. Verifique se são diretamente ou inversamente proporcionais
  3. Monte a proporção com os valores conhecidos
  4. Aplique a fórmula para encontrar o valor desconhecido
  5. Verifique se o resultado faz sentido
